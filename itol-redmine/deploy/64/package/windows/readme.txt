Copy the .wxs and .ico file generated by the installer into this directory.

The installer prints on stdout where the files can be found, e.g.:
Config files are saved to C:\Users\Wolfgang\AppData\Local\Temp\fxbundler4068735248567928189\windows. Use them to customize package.

Insert the following lines at the bottom of the .wxs file inside the <Product> tag. They define the custom actions to register and un-register the Add-in.


    <CustomAction Id='RegisterServer' FileKey='LauncherId' ExeCommand='/RegisterServer user' Return='ignore' />
    <CustomAction Id='UnregisterServer' FileKey='LauncherId' ExeCommand='/UnregisterServer user' Return='ignore' />

    <InstallExecuteSequence>
      <Custom Action='RegisterServer' After='InstallFinalize'>NOT Installed</Custom>
      <Custom Action='UnregisterServer' After='InstallInitialize'>(NOT UPGRADINGPRODUCTCODE) AND (REMOVE="ALL")</Custom>
    </InstallExecuteSequence>
